# 原初引擎与模块

<Column gap="2" fullWidth={true}>

### 总览

<Row gap="20">

<Column gap="2" fullWidth={true}>

> <ItemLink id="gt_shanhai:primordial_omega_engine" />\
> 这台控制器的物品名叫“原始终焉引擎”，这一页里简称原初引擎。它沿用了原始终焉引擎那套主机扫描模块位、模块主动回连主机的框架，但把外观和定位压回了青铜蒸汽体系。\
> 主机成型后会统一管理接入的模块和专用产线，决定它们能处理哪些配方、能跑多快、能不能进入超限状态。\
> 主机四周固定有 16 个模块位，可以装原初模块、太虚熔炼炉、铸币工厂，以及少数专用部件。结构重新成型时，主机会重新扫描这些位置，把已经装好的模块接回主机。

</Column>

<Column gap="2" fullWidth={true}>

> **读图顺序**\
> 先看主机，再看“终焉总览”，最后看“产线模块”。这样能先理解哪一类机器是通用改造，哪一类是专用产线，后面补每个独立页面时也不会重复解释。

</Column>

</Row>

</Column>

<Column gap="2" fullWidth={true}>

### 主机本体

* <ItemLink id="gt_shanhai:primordial_omega_engine" /> 本体自带 5 类基础工序：电炉、组装机、合金炉、锻造锤、打粉机。
* 主机显示为无限并行、无限线程，定位就是把基础工序和外挂模块统一收进一台总控机里。
* 外圈蒸汽机器外壳位可以挂物品输入/输出、流体输入/输出、能源输入，以及维护仓。
* 16 个模块位不用玩家手动绑定，主机成型时会清空旧连接，再按固定位置重新识别。

</Column>

### 运行规则

> * 16 个模块位没有装模块时，要先用蒸汽组装方块占位；需要换模块时，直接把占位换成对应模块就行。
> * 原初引擎和模块都会互相扫描位置。主机重组成型后会重新认模块，模块自己成型后也会主动回连主机。
> * 原初模块通用有 3 类控制位：物质模块槽、线程倍率槽、额外挂载槽。
> * 物质模块槽只负责模块等级和配方条件判定，不是并行槽；很多模块自己的并行倍率，要看它们各自的专用槽位或独立页面。
> * 线程倍率槽负责跨配方线程；放入寰宇并行超限器后，会把模块切到超限模式。
> * 额外挂载槽目前通用支持暗能量倍增器、湮灭核心、超稳态黑洞种子，分别对应降 EU、压缩耗时、处理溢出产物。
> * 如果配置允许，没接主机的模块会显示“独立运行中”；如果没开独立模式，模块就只会待机并提示未连接主机。
> * 如果模块等级、数量或附加条件不够，机器不会硬跑，而是优先提示缺哪一级模块、差多少等效数量。
