# 原初未央重构模块
> <ItemLink id="gt_shanhai:primordial_weiyang_reconstruction_module" />\
> 处理物质异化、梦魇合成、电磁共振、地脉断层与虚空聚流配方。沿用原初引擎的模块并行、线程倍率与无线供电体系。

## 配方类型
* `gtceu:matter_exotic`：物质异化。
* `gtceu:nightmare_crafting`：梦魇合成。
* `gtceu:em_resonance_conversion_field`：电磁共振转化场。
* `gtceu:tectonic_fault_generator`：地脉断层发生器。
* `gtceu:voidflux_reaction`：虚空聚流反应。
