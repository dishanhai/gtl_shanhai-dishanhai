# 商店与钱包系统

<Column gap="2" fullWidth={true}>

### 总览

> 这是一套依附在 <ItemLink id="gt_shanhai:wallet" /> 上的完整交易系统，包含购买/出售商品、货币兑换、自定义兑换和管理员图形化编辑。\
> 结算不是单一货币，而是"星火(数字余额) + 多种GT币 + 背包物品 + AE网络流体"混合成本，一个商品可以同时收好几种。\
> 余额跟着**玩家账户**走，不再存在钱包物品本身上。

</Column>

### 当前范围

* 手持钱包打开商店、浏览分类、购买/出售商品
* 货币中心：单币充值、币种互换、AE抽取入账、币种⇄星火互转
* 兑换中心：作者自定义的任意成本换任意产出清单
* 管理员：编辑权限授予、图形化增删商品、指令式管理、配置文件结构
