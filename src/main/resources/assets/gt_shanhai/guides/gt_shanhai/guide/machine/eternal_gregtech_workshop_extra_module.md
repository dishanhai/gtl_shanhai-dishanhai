# 永恒格雷工坊额外模块

<Column gap="2" fullWidth={true}>

### 总览

> <ItemLink id="gt_shanhai:eternal_gregtech_workshop_extra_module" />\
> 永恒格雷工坊的额外模块。当前源码里它和其他工坊模块一样，接入主机后作为普通模块运行，并只吃主机汇总加成。\
> 这是工坊系列里用于扩展的额外模块位。

</Column>

<Column gap="2" fullWidth={true}>

### 当前行为

* 模块类型为 `extra`。
* 需要连接到永恒格雷工坊主机。
* 当前先作为扩展模块位使用。

</Column>

<Column gap="2" fullWidth={true}>

### 默认参数

> 默认等级 `1`。\
> 默认最大 EUt `42,949,672,940`。\
> 默认 EU 折扣 `0.95`，时长系数 `0.90`，默认并行 `1`，热量 `0`。

</Column>

<Column gap="2" fullWidth={true}>

### 使用建议

* 作为工坊扩展位接入主机后再看整体收益。
* 它目前更偏扩展槽，不是独立逻辑机。
* 如果你后面继续扩永恒工坊系列，这个模块位可以保留。

</Column>
