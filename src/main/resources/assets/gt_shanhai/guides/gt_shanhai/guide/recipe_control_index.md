# 高级配方控制项

这里收纳所有和配方规则管理有关的说明。内容对应 `DShanhaiRecipeModifierAPI` 和 `DShanhaiCommands` 里的实际入口，重点是预设、剥离、替换和删除。

* [配方预设](recipe_preset.md)
* [配方剥离](recipe_strip.md)
* [配方替换](recipe_replace.md)
* [配方删除](recipe_delete.md)

## 当前范围

* 预设文件的保存、加载、覆盖、删除和列表
* 剥离规则的输入/输出、物品/流体、按配方 ID 过滤
* 替换规则的物品替换、流体替换、数量和电路替换
* 按配方 ID 正则删除配方并持久化到规则文件
