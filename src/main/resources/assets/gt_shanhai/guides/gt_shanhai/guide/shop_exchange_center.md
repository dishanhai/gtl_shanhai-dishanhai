# 兑换中心

## 说明

兑换中心是商店界面顶栏的另一个子页面，跟"购买商品"不是一回事：购买是单方向花钱拿货，兑换中心里的每条兑换记录是**作者自定义的"成本→产出"配方**，双方都可以是星火加多种物品/流体的混合组合，而且**支持双向兑换**（正向按配方来，反向把产出和成本对调）。

游戏刚安装时这份清单默认是**空的**，需要管理员自己去配置才会出现兑换条目，具体怎么加见[管理员：授权与配置](shop_admin.md)。

## 怎么兑换

1. 打开兑换中心页面，浏览已配置好的兑换条目。
2. 每条兑换会列出"要付出什么" 和 "能拿到什么"，两边都可能是星火+物品+流体的组合。
3. 确认满足条件后点击兑换，物品部分走背包，流体部分走你绑定的在线AE网络，星火部分走账户余额。
4. 支持反向兑换的条目，可以把产出换回成本，方向不受限制。

## 需要注意

* 涉及流体的兑换，同样需要先绑定好在线的AE提交器。
* 兑换清单是不是有内容、有哪些条目，完全取决于管理员有没有配置过 `exchanges.json`，默认打开可能什么都没有。
