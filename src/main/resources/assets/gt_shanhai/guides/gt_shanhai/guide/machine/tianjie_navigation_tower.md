# 天界领航塔

<Column gap="2" fullWidth={true}>

### 总览

> <ItemLink id="gt_shanhai:tianjie_navigation_tower" />\
> 一座以天界巨构为锚点的广播源多方块。它会把广播半径和配方倍率拉到极高，把周围机器一起卷进天界领航流程。\
> 当前实现里，它是一个很直接的高并行广播塔，不走复杂分支。

</Column>

<Column gap="2" fullWidth={true}>

### 当前行为

* 机器类型为 `tianjie_navigation_tower`。
* 广播半径为 500。
* 最大并行为 32768。
* 固定输出倍率会被推到极高值。

</Column>

<Column gap="2" fullWidth={true}>

### 结构特征

> 结构使用天界主题材料，核心方块和仓室位都比较明确。\
> 玩家视角下，这台机器更像一个大范围的天界引导终端。

</Column>

<Column gap="2" fullWidth={true}>

### 使用建议

* 适合当作大范围引导和倍率放大器看。
* 它的强项是覆盖范围和倍率，不是花哨分支逻辑。
* 放在需要全域推进的天界链路里最顺手。

</Column>
