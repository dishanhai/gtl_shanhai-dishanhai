# 货币中心（ATM）

## 说明

货币中心是商店界面顶栏的一个子页面，专门处理"钱和钱之间"的事情 —— 充值、换币、从AE网络里抽币、以及币种和星火互相转换。跟买卖商品是分开的两套流程。

## 单币充值

把背包里商店认识的实体币（各种GT币物品）按1:1吸进你的账户余额。商店主界面顶栏还有个"充值全部"按钮，效果一样，一次性把背包里所有能识别的币都收进去，省得一种一种点。

## 币种互换

不同币种之间可以互相兑换，换算按**币值表**来算：

```
目标数量 = floor(源数量 × 源币值 / 目标币值)
```

币值表存在 `currency_rates.json` 里，以铜币为1的锚点，按4的幂次往上排阶梯，星门水晶浆币是目前币值最高的一档。想知道两种币之间的换算比例，直接在这个页面里选源币和目标币就能看到。

## AE抽取入账

如果你的AE网络里存着币种物品，货币中心可以直接从网络里抽币直接进账户，不用先搬进背包再充值，前提同样是要有在线绑定的AE提交器。

## 币种⇄星火互转

币种和星火（数字余额）也能互相转换，同样按币值换算比例结算。星火本身不是一种"币"，它只是记在账户上的一个数字，没有对应的实体物品，也没有上限，适合作为跑量交易时的中间媒介。

## 需要注意

* 换算比例由 `currency_rates.json` 决定，管理员改了汇率表，这里显示的比例会跟着变。
* AE抽取同样依赖在线绑定的AE网络，网络离线时抽不到币。
