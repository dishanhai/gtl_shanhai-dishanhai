## 简体中文

### 山海的神人私货 · GT-Shanhai

**GT-Shanhai** 是一款为 Minecraft 1.20.1 与 GregTech Community Edition (GTCEu) 深度定制的工业附加模组，提供强大的高阶机器、自动化系统与无限并行处理能力。

---

#### 🎮 核心功能

**工业机器**
- 精馏塔、热交换器、反应釜等基础工业设备
- 多配方并行处理支持，配方数量无限制

**原初系列模块（16 个）**
- 原初质能核心 — 电能与物质相互转换
- 原初薪火裂解窑 — 集热解炉、裂化炉、高炉、焦炉于一身
- 原初生物核心 — 生物工程与细胞培养
- 原初混沌时空炉 — 物质分解与重组
- 原初因果织矩阵 — 时间与空间操作
- 原初奇点反演核心 — 黑洞与反物质相关处理
- 原初世界碎片收集器 — 跨维度世界碎片采集
- 原初装配线模块 — 流水线式自动化组装
- 原初临界处理模块 — 极限工业加工
- 原初多维爆破核心 — 维度间爆破合成
- 原初超临界物质生成核心 — 超高能态物质制造
- 原初宇宙反应堆 — 宇宙级能源生产
- 原初分子裂隙核心 — 分子级精细加工
- 原初天穹装配核心 — 天文级零件组装
- 原初永恒冶炼炉 — 无损物质冶炼
- 更多模块持续扩展...

**无线供电系统**
- 自动识别并注册全服务器的所有 GTCEu 能源容器机器
- 支持三种分配模式：全网格、半网格半广播、完全广播
- 实时能源分配，无延迟无损耗

**星律样板供料**
- 与 Applied Energistics 2 (AE2) 深度集成
- 支持无限范围、跨维度的自动供料
- 智能缓冲与循环利用

**商店系统**
- 创建自定义玩家商店，支持 NPC 贸易
- 单个商品栏位同时支持多种物品与流体
- 灵活的权限管理与白名单控制

**虚拟物品供应**
- 独立的无限容量虚拟库存系统
- 为超大规模配方提供物资缓冲
- 支持物品与流体混合存储

**数字货币系统**
- 基于玩家 UUID 的星火数字账户
- 支持物品与流体兑换成货币
- 服务器级账户记录与持久化

**并行加速装置**
- 可插拔的模块等级晶体，控制处理速度
- 支持从 64 倍到无限倍的平行处理
- 自动扫描与动态调整

#### 📦 安装要求

**必要依赖：**
- GTCEu 1.4.4+
- GTLCore 1.2.2.8-fix6+
- KubeJS 2001.6.5-build.14+
- Forge 1.20.1+

**可选依赖（功能扩展）：**
- Applied Energistics 2 — 启用星律样板供料
- AE Addon — 增强 AE 集成

---

#### 🚀 快速开始

1. **下载安装**
   ```
   gt_shanhai.jar → mods/
   ```

2. **放置依赖**
   - 确保依赖模组已在 `mods/` 文件夹中
   - 启动游戏，模组自动加载

3. **使用原初模块**
   - 在原始终焉引擎中安装各类模块
   - 配置能源分配模式（质能核心）
   - 插入并行等级晶体以提升处理速度

4. **启用无线供电**
   - 所有能源机器自动加入无线电网
   - 选择合适的分配模式

5. **建立商店**
   - 使用商店编辑工具创建商品
   - 设置玩家权限与交易规则

---

#### 💾 数据保存

所有系统配置均自动保存在服务器存档中：
- 无线供电设备注册表
- 商店信息与玩家账户
- 虚拟物品库存记录
- 玩家数字货币余额

重启服务器后自动恢复。

## English

### GT-Shanhai: A GregTech Community Addon

**GT-Shanhai** is a specialized industrial addon for Minecraft 1.20.1 and GregTech Community Edition (GTCEu), featuring advanced machinery, comprehensive automation systems, and unlimited parallel processing capabilities.

---

#### 🎮 Core Features

**Industrial Machinery**
- Distillery towers, heat exchangers, reactors, and other foundational industrial equipment
- Unlimited recipe parallelization for massive batch processing

**Primordial Modules (16+)**
- **Mass Energy Core** — Convert between energy and matter
- **Flame Cracking Kiln** — 5-in-1 thermal processing (pyrolyse, cracking, blast furnace, lava furnace, coke oven)
- **Biological Core** — Bioengineering and cellular cultivation
- **Chaotic Ephemeral Furnace** — Matter deconstruction and recombination
- **Causal Weaving Matrix** — Time and space manipulation
- **Singularity Inversion Core** — Black hole and antimatter processing
- **World Fragments Collector** — Cross-dimensional fragment harvesting
- **Assembly Line Module** — Automated pipeline assembly
- **Critical Processing Module** — Extreme industrial processing
- **Multidimensional Implosion Core** — Inter-dimensional implosion synthesis
- **Supercritical Matter Generation Core** — Ultra-high energy state matter creation
- **Cosmic Reactor** — Universe-scale power generation
- **Molecular Rift Core** — Molecular-level precision machining
- **Tianqiong Assembly Core** — Astronomical-scale component assembly
- **Eternal Smelting Furnace** — Non-destructive matter smelting
- Plus more modules in active development...

**Global Wireless Power System**
- Auto-detects and registers all GTCEu energy container machines server-wide
- Three distribution modes: Grid-only, Semi-Broadcast, Broadcast-only
- Real-time power distribution with zero latency and loss

**Stellar Pattern Supply**
- Deep integration with Applied Energistics 2 (AE2)
- Infinite-range, cross-dimensional automatic item/fluid feeding
- Smart buffering and resource recycling

**Shop System**
- Create custom player shops with NPC trading support
- Single product slots support multiple item and fluid types simultaneously
- Flexible permission management and whitelist control

**Virtual Item Supply**
- Standalone unlimited-capacity virtual inventory system
- Provides material buffering for massive recipe chains
- Supports mixed item and fluid storage

**Digital Currency System**
- Player UUID-based Spark digital accounts
- Convert items and fluids to currency
- Server-level account records with permanent persistence

**Parallel Acceleration Modules**
- Plug-in-play module-tier crystals controlling processing speed
- Support from 64x to unlimited parallel processing
- Automatic detection and dynamic adjustment

5. **Establish Shops**
   - Use shop editor to create products
   - Set player permissions and trade rules
